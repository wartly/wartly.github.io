What is GDIFY?

GDIFY is a chrome extension that can be used to find GD references on a page.

How to use GDIFY?

When on a website, simply click the pink and green GDIFY icon to highlight all GD references on the page. You can then hover over any reference to see what text referenced it.

How to install GDIFY on chrome?

  1. Unzip the file. Open the zip archive and select the extract all button (or similar) to extract the contents of the file. Inside this new folder there should be another folder called gdify. If done correctly, you should now have a folder named gdify, which contains a folder named gdify, which contains the extension files.
  2. Go to chrome extensions. Go to chrome://extensions.
  3. Turn on developer mode. Ensure the toggle for developer mode is turned on.
  4. Load unpacked. Select "load unpacked" from the developer menu.
  5. Select the nested gdify folder. In the window that has just opened, navigate to where you extracted the zip to. Select the folder named gdify that has a parent folder named gdify.
  6. You are done. If you have done this correctly it will say extension loaded. You can then scroll down until you see it and select the pin icon for easy use.
