At the top of each page, there is a small button that allows you to upload a photo of yourself.
As you scroll, the file rises up.
Scroll some more, and eventually the file will go off screen...
This software works off a theory called Scroll Theory.
Scroll Theory is a theory developed by computer scientist, Wartly
Scroll Theory states that any element that leaves the top of your screen, is actually still moving up in a hypothetical space.
Every website is ran on a server, which is in a specific location.
Although the element is not rendered above your screen, Scroll Theory suggests that it's data and location is still being calculated, in the location of the server.
Even in the location of the server, the viewport is not rendered, but it's data is stored.
This means, that the viewport could be infinitly big.
This is hard to understand, even for us scientists, and that is why this remains just a theory.
Using this theory, we can conclude that by the time you reach the bottom of the 'climbing screen', the file will be a certain height in the sky.
We have calculated the heights of each climbing screen, and they match the heights of each location.
